Data for the different analyses outlined in the paper are stored here, in the respective folders.

- `AraCore_v2_1.wKEGG.mat` is AraCore annotated with KEGG identifiers. Converting to sbml format does not retain the KEGG identifiers in cases where multiple identifiers are mapped to one metabolite, so for the purpose of plugging the pcm into AraCore, this .mat version is used.

- `simulationMedia.tsv` details the bounds of import reactions to simulate phototrophic and heterotrophic growth, respectively
